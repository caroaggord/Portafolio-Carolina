# Portafolio Web - Carolina Aguilar
Este proyecto consiste en el desarrollo de un portafolio web personal como parte de mi formación en desarrollo frontend. Su objetivo es mostrar de manera estructurada mis conocimientos, habilidades y avances en tecnologías web.

## Descripción
El portafolio fue construido utilizando HTML, CSS y JavaScript, integrando buenas prácticas de diseño y organización del código. A lo largo de su desarrollo, se han implementado mejoras progresivas para optimizar tanto la experiencia de usuario como la presentación visual.

## Características
- Estructura semántica con HTML
- Estilos personalizados con CSS
- Uso de Flexbox y Grid para la distribución de elementos
- Diseño responsivo mediante media queries
- Navbar dinámico para navegación en dispositivos móviles
- Imágenes adaptativas
- Sección de intereses interactiva con JavaScript
- Botón flotante de contacto
- Integración de enlaces a GitHub y LinkedIn

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Git y GitHub

## Objetivo del proyecto
Aplicar los conocimientos adquiridos en desarrollo web frontend para construir un portafolio funcional, responsivo y escalable, que permita evidenciar el aprendizaje y evolución en el área tecnológica.

## Estado del proyecto
El proyecto se encuentra en constante mejora, con la intención de seguir integrando nuevos proyectos y funcionalidades conforme avance en mi formación.
